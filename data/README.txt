PlaNet data - Converted from Python Pickles to CSV

- `knowledge-graph-31_7_21.pkl`: our main knowledge graph
- `node_features_armtext.pkl`: node features used in our knowledge graph
- `graph` folder: contains clinical trials data. For example `graph/unique_arms.pkl` provides the information of all trial arms
- `clf_data` folder: contains the classification data used to train and evaluate safety/efficacy prediction models
- `models` folder: contains the pre-trained model (`3u7di6ag`)
- `analysis` folder: contains data used in analysis, such as the pre-trained trial embeddings
